# 42步进电机 + TMC2208 + STM32F103C8T6 接线说明

## 物料清单

| 物料 | 规格 | 数量 |
|------|------|------|
| STM32F103C8T6 | 最小系统板 (中容量, 64KB Flash) | 1 |
| TMC2208 | V3.0 步进驱动模块 | 2 |
| 42步进电机 | 1.8°, 额定电流 ≤1.5A | 2 |
| 电源 | DC 12V~24V, ≥3A | 1 |
| 电解电容 | 100μF / 35V (电源滤波) | 2 |
| 电阻 | 1kΩ (TMC2208 UART 上拉) | 2 |
| 杜邦线 | 母-母 | 若干 |

## 引脚连接表

### Motor X (TMC2208 #1)

| STM32F103C8T6 | TMC2208 引脚 | 说明 |
|---------------|-------------|------|
| **PA0** | STEP | 步进脉冲，上升沿触发 |
| **PA1** | DIR | 方向: L=CW(正转), H=CCW(反转) |
| **PA4** | EN | 使能: L=电机锁定, H=释放 |
| **PA9** (USART1_TX) | PDN_UART | **串 1kΩ 电阻** 后连接 |
| GND | GND | 共地 |
| — | VIO | 接 **3.3V** (不可接5V!) |
| — | VM | 接 12V~24V 电源正极 |
| — | GND | 接电源负极 |

### Motor Y (TMC2208 #2)

| STM32F103C8T6 | TMC2208 引脚 | 说明 |
|---------------|-------------|------|
| **PA6** | STEP | 步进脉冲 |
| **PA7** | DIR | 方向控制 |
| **PB0** | EN | 使能 |
| **PA2** (USART2_TX) | PDN_UART | **串 1kΩ 电阻** 后连接 |
| GND | GND | 共地 |
| — | VIO | 接 **3.3V** |
| — | VM | 接 12V~24V 电源正极 |
| — | GND | 接电源负极 |

### TMC2208 → 42步进电机

| TMC2208 | 电机线圈 | 线色参考 |
|---------|---------|---------|
| A+ | A+ | 红 |
| A- | A- | 蓝 |
| B+ | B+ | 绿 |
| B- | B- | 黑 |

> 线色因品牌各异，用万用表确认线圈导通：A+/A- 一组，B+/B- 一组

## TMC2208 模块设置

### 跳线 / 拨码

| 引脚 | 设置 | 原因 |
|------|------|------|
| MS1 | GND | 微步由 UART 寄存器控制 |
| MS2 | GND | UART 地址 = 0x00 |
| SPREAD | 任意 | 由 UART 配置决定 |

### VREF 电流调节

`Irms = Vref / (0.71 × 2 × Rsense)`

Rsense 典型 110mΩ:
- Vref = 1.0V → Irms ≈ 0.64A
- Vref = 1.2V → Irms ≈ 0.77A

**推荐: VREF 调到 1.0V~1.2V**

> 软件中 `DEFAULT_IRUN=20` 进一步限制，硬件 VREF 是上限

## 电源接线

```
12V~24V 电源
  ├── + ──┬── 100μF 电容 ──┬── TMC2208_X VM
  │       │                └── TMC2208_Y VM
  ├── - ──┬────────────────── TMC2208_X GND
  │       ├────────────────── TMC2208_Y GND
  │       └────────────────── STM32 GND
  │
  └── (STM32 由 USB 或 3.3V 独立供电)
```

> 务必共地！所有模块的 GND 连在一起

## Keil MDK 工程搭建

1. 新建工程 → 选择 STM32F103C8
2. 添加以下文件到工程:
   - `User/main.c`
   - `User/stm32f10x_it.c`
   - `User/StepMotor/bsp_stepmotor.c`
   - `User/TMC2208/bsp_tmc2208.c`
   - `User/MotionPlan/bsp_motionplan.c`
3. 添加库文件 (从已有项目复制):
   - `Libraries/CMSIS/` (core_cm3.c, system_stm32f10x.c, startup)
   - `Libraries/FWlib/src/` (stm32f10x_gpio.c, stm32f10x_rcc.c, stm32f10x_tim.c, stm32f10x_usart.c, misc.c)
4. 添加 include 路径: `User`, `User/StepMotor`, `User/TMC2208`, `User/MotionPlan`, `Libraries/CMSIS`, `Libraries/FWlib/inc`
5. 编译 → 烧录

## 调试步骤

1. 上电后测量 VM 和 VIO 电压是否正常
2. EN 引脚初始为高 (电机释放)，TMC2208_Init 后不变
3. 调用 `Stepper_Enable(&motor, 1)` → EN 拉低 → 电机锁定 (有保持力矩)
4. 调用 `Stepper_MoveTo(&motor, 51200)` → 电机转 1 圈
5. 如果电机不转:
   - 用示波器看 PA9 能否抓到 UART 数据帧 (8 字节)
   - 降低波特率: 把 `TMC2208_BAUDRATE` 改为 9600
   - 检查 STEP 引脚 (PA0) 有无脉冲

## 常见问题

| 现象 | 原因 | 解决 |
|------|------|------|
| 电机不转 | UART 通信失败 | 降波特率到 9600, 检查 1kΩ 电阻 |
| 电机振动不转 | 线圈接错 | 交换 A+/A- |
| 丢步 | 速度/加速度过大 | 降低 max_speed / accel |
| 电机发热 | 电流过大 | 减小 DEFAULT_IRUN (默认20→15) |
